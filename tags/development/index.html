Building My Portfolio Website

Challenges, Choices, and Lessons Learned

Creating a personal portfolio website was something I knew would be useful, but I underestimated just how much went into the process. From choosing the right platform to fine-tuning the design, each step came with its own challenges and learning opportunities.

Choosing Publii & GitHub Pages

Initially, I planned to build the website from scratch—starting with a blank slate and building it piece by piece. However, after reaching a certain point, I realized there might be a more efficient approach. I explored various tools, including Obsidian.md for editing pages, Jekyll (a static site generator), and GitHub Pages for hosting, all with the goal of creating a simple yet personalized URL. Ultimately, I decided to go with Publii—a user-friendly static site generator. This choice allowed me to focus on content creation while still offering customization options. I continued using GitHub Pages for hosting, as it provided a free and reliable way to publish my site (they’re effectively sponsoring the whole project).
